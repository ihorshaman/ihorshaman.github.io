ABOUT ME

Всіх вітаю на своєму сайті, мене звати Шиманович Ігор. Я студент Херсонського державного університету, зараз навчаюсь на третьому курсі. Моя спеціальність - компьютерні науки. Даний сайт является відображенням моїх навичок у WEB розробці.

Я планую стати системним адміністратором у майбутньому, та зробити перекваліфікацію в DevOps. Є невеликий досвід користування Linux дистрибутивів, можу вважатись продвиненим користувачем Windows. Також є невеликий досвід налаштувань віртуальних машин(QEMU) та використання Docker

Цей сайт був зроблений завдяки HTML5, CSS та фреймворку Bootstrap, який спрощує розробку сайтів у десятки разів.
